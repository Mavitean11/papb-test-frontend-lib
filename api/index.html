papb-test-frontend-lib

This is a useless library for the front-end that I develop to experiment with the various Javascript tools available around. The goals:

  • To easily build dists for browsers (perhaps using browserify)?
  • To setup automated releases (perhaps using semantic-release)?
  • To make front-end testing (perhaps using cypress)?
  • To generate and display good documentation in GitHub Pages and/or with now
  • To use greenkeeper and perhaps other good GitHub bots
  • To enforce proper linting (using ESLint)
  • Use TypeScript or at least the so-called "TypeScript typings"

This is a work in progress.
